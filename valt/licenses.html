Valt - Secure Password Manager

Valt is built primarily using Apple's native frameworks. The following Apple frameworks and technologies are used in the app:

Apple CryptoKit

Apple License

Used for AES-256-GCM encryption and cryptographic operations. Provided as part of the iOS SDK.

Apple SwiftData

Apple License

Used for local data persistence and storage management. Provided as part of the iOS SDK.

Apple StoreKit 2

Apple License

Used for in-app purchases and subscription management. Provided as part of the iOS SDK.

Apple LocalAuthentication

Apple License

Used for biometric authentication (Face ID / Touch ID). Provided as part of the iOS SDK.

Swift Argument Parser

Apache License 2.0

Straightforward, type-safe argument parsing for Swift.
